# Demo app
This is an example app, it uses:
  - NestJS
  - CQRS pattern
  - TypeORM
  - Docker
  - PostgreSQL

# To run the application:
  - Go to project root
  `docker-compose up -d`
  - Go to the backend folder
  `yarn run start`
## Useful info
  - Backend runs on `localhost:3000`
  - Todo api `localhost:3000/todo`
  - Swagger `localhost:3000/swagger`
